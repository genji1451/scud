# Отчет по рабочему времени сотрудников

Дашборд для просмотра учета рабочего времени с защитой по логину и паролю.

## Локальный запуск

1. Установите зависимости:
   ```bash
   npm install
   ```

2. Создайте файл `.env.local` с переменными:
   ```
   ADMIN_LOGIN=admin
   ADMIN_PASSWORD=ваш_пароль
   ```

3. Скопируйте `work_summary.json` в папку `public/` (генерируется скриптом `generate_report.py` из корня проекта).

4. Запустите:
   ```bash
   npm run dev
   ```

5. Откройте http://localhost:3000 и войдите с указанным логином и паролем.

## Деплой на Vercel

1. Загрузите проект на GitHub.

2. Подключите репозиторий к [Vercel](https://vercel.com).

3. В настройках проекта Vercel добавьте переменные окружения:
   - `ADMIN_LOGIN` — логин администратора
   - `ADMIN_PASSWORD` — пароль администратора

4. Деплой выполнится автоматически.

## Обновление данных

После запуска `generate_report.py` в корне проекта скопируйте `work_summary.json` в `web/public/` и перезапустите приложение.
